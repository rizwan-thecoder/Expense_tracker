# Expense_tracker
In this project, you will build a command-line-based *Personal Expense Tracker* to help users record and manage their daily expenses. The program will allow users to: 1. Add, View, and Categorize Expenses. 2. Generate Reports on expenses (total spent, category-wise spending, etc.). 3. Store Data in a file (CSV or text) for future use. 4. Visualized
